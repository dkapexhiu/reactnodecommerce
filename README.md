
## Run project

```
git clone
cd reactnodecommerce
npm i
cd client
npm i
cd ..
npm run dev
```

## Project Config

create a `.env` file at root with the following structure.

```
MONGO_URI=
N_EMAIL=
N_PASSWORD=
```

create a `.env` file at client dir root with the following structure.

```
REACT_APP_PS_TOKEN="PAYPAL_SANDBOX_CODE"
```
